1. DATA CRAWLING
In this file there is two main notebook, Weather and Flight Model.
- Notebook_Weather.ipynb used to crawl the weather data. 
- Notebook_FlightModel.ipynb used to crawl the flight model data. (The API expired, need to create a new one)

Foot_Traffic.py
- This is a python code that prompt user with venue name and address. The venue name and address can retrieve from BESTTIME webpage (by specific location).


2. DATA PREPROCESSING

3 Notebook in this Folder
Notebook_Flight - Data preprocessing for flight dataset
Notebook_Weather - Data preprocessing for weather dataset  (the part behind)
Notebook_Merge - Merging dataset

3. EXPLORATORY DATA ANALYSIS
PowerBI.ipynb = The data can be uploaded here, and the data can be sync to Google Sheets and PowerBI can extract data from there

Visualization.pbix - PowerBi visualization of Final Dataset

4. MODEL
FYP.ipynb - All The steps of creating the model


5. PREDICTIVE APPLICATIONS
FlightDisruptionPredictionAPP (FOLDER) - Streamlit APPLICATION 
Prediction.pbix - PowerBi visualization of predicted data.
